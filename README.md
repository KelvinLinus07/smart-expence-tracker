# 💸 Smart Expense Tracker

A behavior-driven expense tracking web application designed not just to record spending, but to **guide users toward smarter financial habits** using AI-based suggestions and gamification.

---

## 🚀 Live Demo

🔗 https://kelvinlinus07.github.io/smart-expence-tracker/

---

## 🧠 Project Vision

Traditional expense trackers only show data.
**Smart Expense Tracker goes beyond that** by acting like a **financial assistant + habit coach**, helping users:

* Understand their spending patterns
* Get personalized suggestions
* Build better financial habits
* Stay motivated through rewards

---

## ✨ Key Features

### 🧾 Expense Tracking

* Add daily expenses easily
* Categorized spending (tea, food, etc.)
* Track remaining daily budget

---

### 🧠 AI Suggestion System

* Detects **main cause of overspending**
* Gives **actionable advice**
* Provides **daily spending plans**
* Uses a **friendly coaching tone**

📌 Example:

> “You crossed your daily budget mainly because of tea. Try avoiding it for the rest of the day.”

---

### 🎮 Gamification System

* Earn badges for good financial behavior
* Encourages consistency and discipline

🏆 Example Badges:

* 💰 Saver Badge
* 🔥 Streak Master
* 🎯 Goal Crusher

---

### 📊 Smart Budget Logic

* Monthly budget allocation
* Fixed expense calculation
* Dynamic daily limit generation

---

## 🧩 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Storage:** LocalStorage
* **Visualization:** Custom UI (No external frameworks)
